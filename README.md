# Bike-Sales-Dashboard-Excel
## Description
An interactive Excel dashboard developed to analyze customer demographics (Income, Age Group, Region) and their impact on bike purchasing trends.

## Key Features:
  - Used Pivot Tables for data aggregation.
  - Implemented Slicers for interactive filtering (Region, Age Group, etc.).
  - Data cleaning including removing duplicates and correcting categories.

---

## Analysis Workflow
1. Data Cleaning:
   - Removed duplicate entries to ensure data integrity.
   - Standardized "Marital Status" and "Gender" columns for better grouping.
   - Created "Age Brackets" using nested IF statements to segment customers into Adults, Middle-Aged, and Seniors.
2. Data Modeling: Built multiple Pivot Tables to aggregate average income and purchase counts.
3. Visualization:
   - Designed a user-friendly interface with Slicers for dynamic filtering.
   - Used Line and Bar charts to compare regional and demographic performance.

---

## Key Insights
  - **Income Impact**: Higher average income correlates with a higher likelihood of bike purchases, particularly in the "Middle Age" segment.
  - **Regional Trends**: North America shows the highest volume of purchases, with a notable preference among male customers.
  - **Commute Factor**: Customers with shorter commute distances (0-1 miles) are more likely to purchase a bike compared to long-distance commuters.

---

## Tools Used
Microsoft Excel: Pivot Tables, Slicers, Data Cleaning (Power Query or Filter/Sort), and Advanced Formulas.
